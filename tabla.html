Introducción a la Web

Conceptos básicos de la Web
Término Definición Ejemplo
HTML Lenguaje de marcado para crear la estructura de las páginas web. <h1>Título</h1>
CSS Lenguaje para diseñar y estilizar las páginas web. color: red;
URL Dirección de una página web en Internet. https://www.ejemplo.com
HTTP Protocolo de transferencia de hipertexto para la comunicación en la web. http://
